## Setting Up the Project

To run this app, you need to provide your own Google Maps API key.

1. Create a `config.js` file in the root of the project.
2. Add the following code to `config.js`:

   const API_KEY = "your_google_maps_api_key_here";
   export { API_KEY };
3. Make sure you replace your_google_maps_api_key_here with your actual Google Maps API key.
4. Save the file and run the app.
